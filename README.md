# CSE341-team

## Setup
To run this project, install it locally using npm or pnpm:

```
$ npm install
$ npm start
or
$ pnpm install
$ pnpm run dev
```

## Routes
```
http://localhost:8080/spell
http://localhost:8080/api-docs

http://localhost:8080/weapons/all
http://localhost:8080/weapons/create
```
